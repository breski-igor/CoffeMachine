# CoffeeMachine

## Description
CoffeeMachine is a C++ application that simulates a coin-operated coffee vending machine.
The application loads data from an XML file, allows the user to view product and coin stock, and order a product.
After execution, updated data is saved back to the XML file.

## Features
- Loads product and coin data from an XML file  
- Displays available products and coin stock  
- Allows users to insert coins and order a product  
- Calculates change and updates stock accordingly  
- Saves updated data back to the XML file
